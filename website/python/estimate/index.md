## 布尔类型和运算符

```js
bool_1 = True;
bool_2 = False;

// == != > < >= <=
```

## if 语句的基本形式

```js
if 语句判断：
    条件成立，执行逻辑
```

## if else 语句

```js
if 语句判断：
    条件成立，执行逻辑
else:
    执行逻辑
```

## if elif else 语句

```js
if 语句判断：
    条件成立，执行逻辑
elif:
    执行逻辑
elif:
    执行逻辑
  ...
else:
    执行逻辑
```

## 判断语句的嵌套

- 代码逻辑执行，会有多层的判断语句

```js
if 语句判断：
    条件成立，执行逻辑
    if 语句判断：
        条件成立，执行逻辑
    else:
        执行逻辑
else:
  执行逻辑
```
