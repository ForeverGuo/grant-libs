### 引入

```js
import { getMonth } from "grantlibs";
或;
const { getMonth } = require("grantlibs");
```

```ts

getMonth(): string

```

### 参数

no params

number: 当前月份。

### 示例

```js
const month = getMonth();
// => '03'
```
