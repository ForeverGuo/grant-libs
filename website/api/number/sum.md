### 引入

```js
import { sum } from "grantlibs";
或;
const { sum } = require("grantlibs");
```

### 使用

```ts

sum(number: number[]): number

```

### 参数

- number: 数值数组

### 返回

number

### 示例

```js
sum([1, 2, 3, 4, 5]);
// => 15
```
