### 引入

```js
import { average } from "grantlibs";
或;
const { average } = require("grantlibs");
```

<div style="color: #E6A23C; fontSize: 18px; padding: 20px 0">
  数值求平均值
</div>

### 使用

```ts

average(number: number[]): number

```

### 参数

- number: 数值数组

### 返回

number

### 示例

```js
average([1, 2, 3, 4, 5]);
// => 3
```
