### 引入

```js
import { getSecond } from "grantlibs";
或;
const { getSecond } = require("grantlibs");
```

### 使用

```ts

getSecond(): string

```

### 参数

无

### 返回

string: 秒

### 示例

```js
const second = getSecond();
// => 10
```
