### Start

```shell

  pnpm install grantguo-libs -D 或 yarn add grantguo-libs -D

  // es
  import { calendar, store } from 'grantguo-libs'
  const year = calendar.getYear()

  // cjs
  const { calendar, store } = require('grantguo-libs')
  const year = calendar.getYear()

```
